Проект "Автомат по продаже напитков"
=====================================
Позволяет вносить сумму с помощью кнопок-монет, выбирать от одного до пяти напитков и забирать их вместе со сдачей.
Закончившиеся напитки в ассортименте не показываются, до внесения необходимой суммы выбрать напиток невозможно.
Заблокированные монеты подсвечены серым цветом, нажатие на них ни к чему не приведет.

Административная панель находится на второй странице, доступ в нее возможен только по секретному ключу - 
параметру маршрута (localhost:*/admin/koko). В случае несовпадения ключа - перенаправление обратно
на страницу с автоматом.
Панель имеет два режима:
1. Конфигурации автомата - настройка блокировки монет и баланса автомата
2. Настройка ассортимента - добавление, редактирование и удаление напитков в автомате
======================================

Использованные технологии: Angualr 6(клиентская часть) + .NET Core Web.API c MS SQL Server Express(серверная)
Файл .mdf с базой данных лежит в папке App_Data, желательно ее заимпортить в Express-экземпляр сервера, 
тогда не нужно менять строку подключения (переменная con метода ConfigureServices в файле Startup.cs)

Примечание: Bootstrap использовался в админ панели только ради быстрого дизайна.
Большая просьба прислать фидбэк, независимо от результата :)
